# 💱 Валютный дашборд

Интерактивный дашборд для отслеживания курсов валют всех стран. Построен с использованием Streamlit.

## 🚀 Функционал

- 📊 Отображение актуальных курсов валют
- 📈 Визуализация топ-10 самых дорогих валют
- 💱 Конвертер валют
- 📅 Автоматическое обновление данных
- 🎨 Адаптивный интерфейс

## 🛠 Технологии

- Python 3.9+
- Streamlit
- Pandas
- Plotly
- Requests
